# 🌐 Portal de Projetos - Sistemas para Internet (UEG)

Este repositório contém o desenvolvimento de um **portal web simples** criado como atividade de estudo do grupo de projetos da **UEG - Sistemas para Internet**.

O objetivo do portal é apresentar oportunidades de projetos e materiais relacionados às iniciativas do grupo, permitindo que estudantes conheçam e participem das atividades propostas.

---

## 📚 Objetivo da Atividade

A atividade consiste em **criar e publicar um portal na web utilizando apenas ferramentas gratuitas**, seguindo as orientações do professor.

O portal deve:

* Ser publicado no **GitHub**
* Utilizar apenas **ferramentas gratuitas ou versões free**
* Apresentar pelo menos o conteúdo do documento **"Oportunidades de Projetos"**
* Ser desenvolvido **individualmente**, para que cada aluno compreenda todo o processo de criação e publicação de um portal

Também é permitido utilizar **ferramentas de Inteligência Artificial**, desde que o aluno compreenda o código gerado.

---

## 🛠 Tecnologias Utilizadas

O portal foi desenvolvido utilizando tecnologias básicas da web:

* **HTML5** – estrutura do site
* **CSS3** – estilização e layout
* **GitHub** – versionamento de código
* **GitHub Pages** – hospedagem gratuita do portal

---

## 📄 Conteúdo do Portal

O portal apresenta informações sobre possíveis projetos de extensão e pesquisa, incluindo:

* 🗺️ **Mapeando a Cidade**
  Projeto de construção colaborativa de mapas digitais.

* 🤖 **Chatbot com IA para Educação**
  Desenvolvimento de chatbot educacional integrado à plataforma **LAPIG na Escola**.

Os documentos completos podem ser acessados diretamente no portal.

---

## 🚀 Publicação do Portal

O site pode ser publicado gratuitamente utilizando **GitHub Pages**.

Passos básicos:

1. Criar um repositório no GitHub
2. Enviar os arquivos do projeto
3. Ativar o **GitHub Pages**
4. Acessar o portal pelo link gerado

---

## 🎯 Objetivo Educacional

Este projeto tem como objetivo:

* Praticar **desenvolvimento web básico**
* Aprender **versionamento com Git e GitHub**
* Entender o processo de **publicação de um site**
* Incentivar a participação em **projetos acadêmicos e de extensão**
