🌐 Portal de Projetos - Sistemas para Internet (UEG)

Este repositório contém um portal web simples desenvolvido como atividade acadêmica do curso de Sistemas para Internet da UEG.

O portal apresenta algumas oportunidades de projetos e materiais de estudo para estudantes interessados em participar das iniciativas do grupo.

📚 Objetivo

Praticar o processo completo de criação e publicação de um site:

Desenvolvimento de páginas web

Uso de Git e GitHub para versionamento

Publicação gratuita de um site na internet
